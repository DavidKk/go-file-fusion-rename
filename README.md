## 使用说明

1. 下载对应平台的版本并解压
2. 运行程序，会出现在系统托盘
3. 右键托盘图标选择转换方向（简转繁/繁转简）
4. 点击"选择文件"选择要转换的文件或文件夹
5. 确认后自动完成文件名和内容的转换
